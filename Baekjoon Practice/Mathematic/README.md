1. ``나머지 연산``
```
수열의 나머지가 같은지 검사하기..
s[i] = m*a + r
s[i-1] = m*b + r
if(나머지가 같다면)
s[i] - s[i-1] = m*(a-b) +(r-r)
이므로,
s[i] - s[i-1] = m * (a-b)이다.
```
